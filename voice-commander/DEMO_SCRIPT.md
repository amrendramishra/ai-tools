# Voice Commander Demo Video Script (for YouTube)

## Title: "I Control My Mac with Voice — 45 AI Tools with One Command"

## Duration: 3-5 minutes

### Scene 1: Hook (0-15s)
- Show terminal, type: `vc`
- Say: "Hey Jarvis, generate content for all my YouTube channels"
- AI responds and executes

### Scene 2: What is Voice Commander? (15-45s)
- Show the help menu
- Explain: "I built 45 AI tools and one voice interface to control them all"
- Show quick commands: status, time, focus

### Scene 3: Demo Commands (45s-3min)
- "research quantum computing" → shows Tavily research
- "write email about project update" → drafts email, copies to clipboard
- "start focus 25" → starts Pomodoro
- "generate content" → creates YouTube scripts
- "code review" → reviews git diff
- "summarize" → summarizes clipboard
- "open web ui" → opens local ChatGPT

### Scene 4: How It Works (3-4min)
- Show architecture: Voice → Intent → Ollama → Execute
- Show Ollama running: `ollama list`
- Show models: 6 models, 18GB, all free

### Scene 5: CTA (4-5min)
- "All 45 tools are open source"
- Show GitHub: github.com/amrendramishra/ai-tools
- "Link in description. Star the repo if you found this useful!"

## Recording Tips:
- Use QuickTime screen recording
- Record terminal in full screen
- Use a dark theme
- Speak clearly, show results
