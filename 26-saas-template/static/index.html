API Documentation

POST/api/generate

Generate text from a prompt. Send {"prompt": "your text"} with X-API-Key header.

POST/api/summarize

Summarize text. Send {"text": "...", "max_length": "short|medium|long"}.

POST/api/translate

Translate text. Send {"text": "...", "target_language": "Spanish"}.

GET/api/usage

Get usage statistics for your API key.

GET/health

Health check - no authentication required.
