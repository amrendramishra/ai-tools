# Trending Topic Detector

AI-powered trending topic finder for YouTube channels. Scrapes Google Trends and Twitter/X, then uses Ollama to rank topics by virality potential, competition, and channel relevance.

## Requirements

- Python 3.11+ (venv at `~/Downloads/AI/.venv`)
- Ollama running at `localhost:11434`
- Model: `llama3.2` (configurable with `--model`)

## Dependencies

```bash
source ~/Downloads/AI/.venv/bin/activate
pip install requests beautifulsoup4
```

## Usage

### By Niche
```bash
# Find trending topics for a niche
./trending_detector.py --niche "technology/tech news"
./trending_detector.py --niche "horror stories"
./trending_detector.py --niche "finance/money tips"
```

### By Channel
```bash
# Suggest topics for a specific channel
./trending_detector.py --channel tech_in_5_hindi
./trending_detector.py --channel horror_ki_kahani
./trending_detector.py --channel money_in_5
```

### Daily Scan
```bash
# Run daily scan for ALL 9 channels
./trending_detector.py --daily
```

## Options

| Flag | Description |
|------|-------------|
| `--niche, -n` | Filter topics by content niche |
| `--channel, -c` | Suggest topics for specific channel |
| `--daily, -d` | Run daily scan for all channels |
| `--model, -m` | Ollama model (default: llama3.2) |

## Data Sources

1. **Google Trends RSS** - India-specific trending searches with traffic data
2. **Twitter/X Trends** - Scraped from aggregator sites (getdaytrends.com, trends24.in)
3. **AI Suggestions** - Original topic ideas generated by Ollama

## AI Ranking Criteria

Topics are scored on:
- **Virality Score** (40% weight): How likely to go viral
- **Competition Score** (30% weight): Lower competition = easier to rank
- **Relevance Score** (30% weight): How well it fits the channel niche

## Output Format

Results are saved as JSON in `trends/`:

```
trends/
├── 2026-07-10_tech_in_5_hindi_1430.json
├── 2026-07-10_all_1500.json
└── 2026-07-10_daily_report.json
```


## Integration with Content Pipeline

This detector works with Project 06 (AI Content Pipeline):
```bash
# Find trending topic, then generate content
./trending_detector.py --channel gyaan_in_5
# Copy a top topic, then:
cd ../06-content-pipeline
./content_pipeline.py --channel gyaan_in_5 --topic "your trending topic here"
```

Or use the content pipeline's `--auto` flag which has built-in topic generation.
